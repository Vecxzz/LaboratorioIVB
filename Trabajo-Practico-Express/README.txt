Trabajo Práctico Express

Este proyecto consiste en una aplicación React servida mediante un servidor creado con Express. El servidor se encarga de mostrar el contenido estático generado por React.

PASOS PARA EJECUTAR

1. Instalar dependencias en la app de React
cd proyecto-React
npm install

2. Instalar dependencias en el servidor
cd servidor
npm install

3. Correr el servidor
node server.js

4. Abrir el navegador y entrar a: http://localhost:3000
